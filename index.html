Some of my projects

Sentiment Analysis on tweets

I developed a sentiment analysis model for Arabic social media by fine-tuning a pre-trained AraBERT model using the Hugging Face transformers library on a public dataset of Arabic tweets. This model achieved an F1 score of 80% on the validation set, demonstrating strong generalization and the effective application of state-of the-art NLP techniques.

Summarizing text into key points

I fine tuned Gemma-7b-it model using Low-Rank Adaptation (LoRA) to enhance text summarization capabilities efficiently. The fine-tuning process focused on optimizing the model to produce accurate and concise summaries across diverse text samples. Performance was evaluated using the ROUGE metric..

Predict future orders based on historical data

Developed a machine learning model to forecast orders for an e-grocery store using historical data as part of a Kaggle competition. I used tree-based models, including LightGBM, CatBoost, and HistGradientBoost, with a weighted ensemble and root mean squared error as metric. Applied advanced data preprocessing and feature engineering to align with real-world demand forecasting requirements.

Finds and labels locations in text.

I fine-tuned a RoBERTa model to identify and extract location names from text. This project was a ML competition in Zindi platform hosted by Microsoft, where I achieved a silver medal.
